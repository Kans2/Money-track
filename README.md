# MoneyTrack - Personal Finance Manager

A comprehensive personal finance management application built with React and Tailwind CSS. Track your income, expenses, and manage your budget effortlessly.

## Features

- **Dashboard**: Overview of current balance, income, and expenses.
- **Transaction Management**: Add, edit, and delete transactions.
- **Category Management**: Customize income and expense categories.
- **Reports**: Visual charts and insights into spending habits.
- **Filters**: Filter transactions by date range, type, and category.
- **SMS Parsing**: Automatically parse transactions from SMS messages.
- **Dark Mode**: Toggle between light and dark themes.

## Installation

1. Clone the repository:
   ```bash
   git clone <repository-url>
   cd money
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Run the development server:
   ```bash
   npm run dev
   ```

## Usage

- Open [http://localhost:5173](http://localhost:5173) to view the app.
- Use the **+** button to add new transactions.
- Click the **Filter** icon in the header to filter transactions.
- Navigate between **Transactions**, **Categories**, and **Reports** tabs.

## Technologies Used

- **React**: Frontend library
- **Tailwind CSS**: Styling framework
- **Chart.js**: Data visualization
